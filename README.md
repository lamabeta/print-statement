# Print Statement Project

### Objective
The project's main focus is on understanding how to print simple output in Java.

### Problem
Print your name and age along with some messages.

### Implementation
- Print a welcome message.
- Print your name and age.
- Print  your age.

```java
public class Main {
    public static void main(String[] args) {
        /* add your code here */

    }
}
```

### Expected output
```
Welcome to the program!
My name is Sari.
I am 13 years old.
```
